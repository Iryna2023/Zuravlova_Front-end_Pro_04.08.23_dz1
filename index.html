Git Bash. Команди:

  • $ git status - перегляд змін та створення коммітів (фіксація змін)
  • $ git add [файл] - індексує вказаний файл для наступного коміту
  • $ git commit -m "[повідомлення з описом]" - фіксує проіндексовані зміни та зберігає їх в історію версій
  • $ git push [віддалений репозиторій] [гілка] - Завантажує всі зміни локальної гілки у віддалений репозиторій
